HTML页面的名称

名称的主要作用

  • 告知用户网页描述的主要内容
  • 被浏览器用来创建书签
  • 被搜索引擎用来建立页面索引

名称的设置方法

标签<title>可为HTML页面设置一个名称,该名称将出现在浏览器的标题栏中.
示例代码,点击链接可查看title.html

如图所示,该代码创建了一个标题名称为”An example of title”的网页.

有关名称的注意事项

  • 每个页面只能有一个名称,该名称只能包含纯文本
  • 尽可能使用描述性强的简短的名称,使其通俗易懂,这样能确保用户理解网页的主要内容
  • 标签<title>可包含任意长度的文本,但显示时可能会被浏览器截短
  • 标签<title>中不能包含其他标签

标题

标题用于为网页各部分添加名称.HTML中共有6级标题,分别用标签<h1>到<h6>表示.级别数字越小,标题越大.
